## Tech Stack

- **Framework:** [Jekyll](https://jekyllrb.com/) — static site generator
- **Hosting:** [GitHub Pages](https://pages.github.com/)
- **Template:** Based on [academic-homepage](https://github.com/luost26/academic-homepage) by [luost26](https://github.com/luost26)
- **Languages:** HTML/CSS, JavaScript, Liquid (Jekyll templating)
- **Features:** Bilingual support (Chinese / English), responsive design, publications showcase, news feed

## Running Locally

```bash
# Install dependencies
bundle install

# Start local development server
bundle exec jekyll serve

# Browse to http://localhost:4000
```
